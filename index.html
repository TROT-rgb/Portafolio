Proyectos

"Dead by Daylight Characters" es un sitio web que he creado completamente por mí mismo, inspirado en la página oficial de un juego de terror. Este sitio fue desarrollado para la empresa "Vivencia Baires" como una forma de práctica para obtener una posición en dicha empresa. Las instrucciones para el desarrollo incluían la creación de un sitio web desde cero, la integración de una conexión con carteras de criptomonedas, y el uso de tecnologías como React y Node. La página es muy fácil de usar y presenta información sobre el juego, opciones de compra, dos secciones que muestran los personajes del juego, y descripciones breves de cada uno de ellos.

En este proyecto fui integrado a un equipo programadores para desarrollar la página de "AMD Red League". Este sitio web es para el equipo oficial de deportes electrónicos de AMD, y fui asignado por medio de la empresa Vivencia Baires.
